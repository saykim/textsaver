# Text Saver

"Text Saver"는 반복적으로 사용하는 텍스트 스니펫, 코드 조각, 프롬프트 등을 손쉽게 저장하고 웹 페이지 어디에서나 `//` 단축키로 빠르게 불러와 자동 완성할 수 있는 크롬 확장 프로그램입니다.

## 🚀 주요 기능

- **✍️ 텍스트 스니펫 관리**: 제목, 내용, 태그를 포함한 텍스트를 저장, 수정, 삭제할 수 있습니다.
- **⭐ 북마크**: 자주 사용하는 중요한 텍스트를 북마크하여 별도로 관리할 수 있습니다.
- **🔍 강력한 검색**: 저장된 모든 텍스트를 제목, 내용, 태그 기준으로 빠르게 검색할 수 있습니다.
- **⚡ `//` 자동 완성**: 웹 페이지의 입력창(textarea, input 등)에서 `//`와 함께 키워드를 입력하면, 저장된 텍스트를 검색하고 선택하여 즉시 삽입할 수 있습니다.
- **🖱️ 컨텍스트 메뉴로 저장**: 웹 페이지에서 텍스트를 선택하고 마우스 오른쪽 버튼을 클릭하여 'Pallet로 선택 내용 저장' 메뉴를 통해 해당 텍스트와 페이지 URL을 바로 저장할 수 있습니다.
- **⌨️ 단축키 지원**: `Cmd+Shift+T` (Mac) 또는 `Ctrl+Shift+T` (Windows) 단축키로 `//` 자동 완성 기능을 빠르게 켜고 끌 수 있습니다.
- **💾 데이터 관리**: 저장된 모든 데이터를 JSON 파일로 내보내거나, 백업 파일로부터 데이터를 가져올 수 있습니다. 또한, 모든 데이터를 초기화하는 기능도 제공합니다.
- **🤖 기본 프롬프트 제공**: CoT, ToT, ReAct 등 유용한 프롬프트 엔지니어링 템플릿이 기본으로 제공되며, 언제든지 복원할 수 있습니다.
- **📝 자동 저장**: 팝업창에서 텍스트 작성 시 내용이 자동으로 임시 저장되어, 예기치 않게 창이 닫혀도 데이터를 보호합니다.

## 📖 사용 방법

### 1. 텍스트 저장하기
- 확장 프로그램 아이콘을 클릭하여 팝업을 엽니다.
- '저장' 탭에서 제목, 내용, 태그(쉼표로 구분)를 입력하고 '저장하기' 버튼을 클릭합니다.

### 2. `//` 자동 완성 사용하기
1. 웹사이트의 텍스트 입력창에 `//`를 입력합니다.
2. 이어서 찾고 싶은 텍스트의 키워드를 입력하면 검색 결과 UI가 나타납니다. (예: `//분석`)
3. 키보드 화살표(↑, ↓)로 원하는 항목을 선택하고 `Enter` 키를 누르면 해당 텍스트의 내용이 입력창에 자동으로 삽입됩니다.
4. `Esc` 키를 누르면 UI가 닫힙니다.

### 3. 웹 페이지에서 직접 저장하기
1. 웹 페이지에서 저장하고 싶은 텍스트를 마우스로 선택합니다.
2. 마우스 오른쪽 버튼을 클릭하여 나타나는 메뉴에서 'Pallet로 선택 내용 저장'을 선택합니다.
3. 선택한 텍스트가 내용으로, 페이지의 URL과 제목이 메타데이터로 함께 저장됩니다.

### 4. 데이터 관리하기
- '설정' 탭에서 `//` 자동 완성 기능의 사용 여부를 설정할 수 있습니다.
- '데이터 내보내기' 버튼으로 모든 데이터를 JSON 파일로 백업할 수 있습니다.
- '데이터 가져오기'로 백업 파일을 복원할 수 있습니다.

## 🛠️ 로컬 환경에서 설치 및 테스트

1. 이 저장소를 로컬 컴퓨터에 복제(clone)합니다.
   ```bash
   git clone https://github.com/your-username/text-saver-extension.git
   ```
2. Chrome 브라우저를 열고 주소창에 `chrome://extensions`를 입력하여 확장 프로그램 관리 페이지로 이동합니다.
3. 오른쪽 상단의 '개발자 모드(Developer mode)'를 활성화합니다.
4. '압축 해제된 확장 프로그램을 로드합니다(Load unpacked)' 버튼을 클릭합니다.
5. 파일 탐색기에서 이 프로젝트 폴더를 선택하여 로드합니다.
