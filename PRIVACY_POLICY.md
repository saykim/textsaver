# Text Saver 개인정보 처리방침

*최종 수정일: 2024년 7월 24일*

본 개인정보 처리방침은 Text Saver 크롬 확장 프로그램(이하 "확장 프로그램")의 사용자 데이터 수집, 사용, 관리에 관한 정책을 설명합니다.

## 1. 수집하는 정보

본 확장 프로그램은 다음과 같은 정보를 수집합니다:

- 사용자가 입력한 텍스트 및 제목
- 사용자가 지정한 태그
- 저장 날짜 및 시간
- 북마크 상태

**중요**: 모든 데이터는 오직 사용자의 브라우저 내 Chrome 로컬 저장소에만 저장되며, 외부 서버로 전송되지 않습니다.

## 2. 데이터 사용 목적

수집된 데이터는 다음과 같은 목적으로만 사용됩니다:

- 사용자가 저장한 텍스트의 표시 및 관리
- 텍스트 내용과 태그 기반 검색 및 필터링
- 북마크 기능 제공
- 사용자의 클립보드 복사 기능 지원

## 3. 데이터 저장 방식

- 모든 데이터는 Chrome의 `storage.local` API를 통해 사용자의 브라우저에만 저장됩니다.
- 데이터는 브라우저의 보안 정책에 따라 보호됩니다.
- 데이터는 다른 웹사이트나 확장 프로그램에서 접근할 수 없습니다.

## 4. 사용자 권한

본 확장 프로그램은 기능 작동을 위해 다음과 같은 브라우저 권한을 요청합니다:

- `storage`: 텍스트 데이터를 로컬에 저장하기 위해 필요
- `clipboardWrite`: 텍스트를 클립보드에 복사하기 위해 필요
- `downloads`: 데이터 내보내기 기능을 제공하기 위해 필요

## 5. 데이터 공유

- 본 확장 프로그램은 어떠한 형태로도 사용자 데이터를 제3자와 공유하지 않습니다.
- 분석이나 마케팅 목적으로 사용자 데이터를 수집하지 않습니다.
- 외부 서버와 통신하지 않으므로 데이터 유출 위험이 없습니다.

## 6. 데이터 관리 및 사용자 권리

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

- **데이터 내보내기**: 확장 프로그램의 '설정' 탭에서 모든 저장된 데이터를 JSON 파일로 내보낼 수 있습니다.
- **데이터 가져오기**: 이전에 내보낸 데이터를 다시 가져올 수 있습니다.
- **데이터 삭제**: 개별 텍스트를 삭제하거나 '설정' 탭에서 모든 데이터를 초기화할 수 있습니다.
- **확장 프로그램 제거**: Chrome에서 확장 프로그램을 제거하면 모든 관련 데이터가 삭제됩니다.

## 7. 개인정보 처리방침 변경

본 개인정보 처리방침이 변경될 경우, 업데이트된 내용은 이 문서에 반영됩니다. 주요 변경사항이 있을 경우 확장 프로그램을 통해 사용자에게 알립니다.
